C++ GUI Calculator 

A simple GUI Calculator built using C++ and Windows API (Win32) in Visual Studio 2026 Community Edition.
This project demonstrates basic GUI programming, button handling, and arithmetic operations.

Features
Graphical User Interface
Basic arithmetic operations
Addition (+)
Subtraction (−)
Multiplication (×)
Division (÷)
Clear (C) button
Read-only display box
Equal-sized buttons with proper spacing
Custom color theme

Technologies Used
Language: C++
GUI Framework: Windows API (Win32)
IDE: Visual Studio 2026 Community
Platform: Windows

How to Run the Project

1. Open Visual Studio 2026 Community
2. Create a Win32 Desktop Application
3. Copy the code from calculator.cpp
4. Paste it into the main .cpp file
5. Build and run the project
